wccresponsive.theme Installation
--------------------------------

To install wccresponsive.theme using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``wccresponsive.theme`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        wccresponsive.theme

* Re-run buildout, e.g. with:

    $ ./bin/buildout
